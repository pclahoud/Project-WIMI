Define Dimensions

Create independent categories for classifying your questions. Each dimension represents a different way to categorize the same question.

💡 Example: NBME Shelf exams use 3 dimensions: Site of Care (where), Physician Task (what), and System (which organ system). Each question gets tagged in all three dimensions.
Dimension Options:
  • Required — User must tag every question in this dimension. Can't skip it.
  • Allow multiple selections — User can pick more than one option when tagging (checkboxes instead of radio buttons).

Weight Calculation Settings

Configure how topic weights are calculated and balanced across your subject hierarchy.

Autonomous Weight Balancing

When enabled, editing one topic's weight automatically adjusts sibling weights to maintain 100%. When disabled, you'll manually balance weights and see both relative and absolute percentages.

Weight Precision

How precise should weight percentages be? Higher precision allows finer control.

Balancing Algorithm

Proportional: Distributes changes based on current sibling weights (preserves relative importance).
Even: Distributes changes equally across all siblings (simpler behavior).

Simple Exam Structure

Best for exams with straightforward topic organization. Questions are categorized in a single hierarchical tree, making it easy to track progress by topic and subtopic.

Recommended For:

  • SAT Math (Algebra, Geometry, etc.)
  • GRE Verbal (Reading Comprehension, Vocabulary, etc.)
  • LSAT Logic Games
  • Subject tests with clear topic hierarchies

Example Structure:

Mathematics ├─ Algebra │ ├─ Linear Equations │ └─ Quadratic Equations └─ Geometry ├─ Triangles └─ Circles

Analytics Include:

  • Performance by topic (sunburst, treemap)
  • Mistake type distribution
  • Time vs difficulty analysis
  • Goal tracking and streak management
💡 Tip: If you're unsure, start with a Simple exam. You can convert to Multi-Dimensional later if needed.

Multi-Dimensional Exam Structure

Best for complex standardized exams where questions can be categorized in multiple independent ways simultaneously.

Recommended For:

  • NBME Shelf Exams (Internal Medicine, Surgery, etc.)
  • USMLE Step 1, Step 2 CK
  • MCAT (with multiple content categories)
  • Any exam with official content outlines using multiple dimensions

Example: NBME Shelf Exam

Each question is tagged across 3 independent dimensions:

Question: "A 68-year-old presents to clinic with palpitations..." Tags: ├─ Site of Care: Ambulatory ├─ Physician Task: Diagnosis └─ System: Cardiovascular → Arrhythmias

Advanced Analytics Include:

  • Everything from Simple exams, plus:
  • Cross-dimension heatmaps (e.g., Site × Task performance)
  • Interaction effect detection ("You struggle with Cardio specifically in Emergency settings")
  • Triple-dimension analysis
  • Context-aware study recommendations
💡 Tip: Multi-dimensional exams unlock powerful analytics but require tagging questions in multiple categories. Best for exams with official content outlines that use multiple categorization systems.
