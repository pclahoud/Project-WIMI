REFLECTION

I confused the mechanism of action between ACE inhibitors and ARBs. I knew that both affect the renin-angiotensin system, but I mixed up which one blocks the receptor vs. which one inhibits the enzyme. I need to create a clear mental model distinguishing these two drug classes.

EXPLANATION

ACE inhibitors (like lisinopril) block angiotensin-converting enzyme, preventing conversion of angiotensin I to angiotensin II. ARBs (like losartan) block the AT1 receptor where angiotensin II acts. Key mnemonic: ACE = Enzyme, ARB = Receptor Blocker. Both reduce BP but via different mechanisms.
